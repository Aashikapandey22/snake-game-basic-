# Snake Game Deluxe Web Edition

Snake Game Deluxe Web Edition is a browser-based arcade game developed using **HTML, CSS, and JavaScript**.  
The project recreates the classic Snake game with additional features such as a leaderboard system, obstacles, pause/resume functionality, and a forum section for user interaction.

The game runs completely in the browser without requiring any backend server.

---

## Project Objective

The objective of this project is to develop an interactive and responsive snake game that allows users to:

- Play the classic snake game in a web browser
- Track scores in real time
- Compare scores through a leaderboard
- Interact with other players using a forum/comment section

---

## Technologies Used

- **HTML5** – Structure of the web application
- **CSS3** – Styling and layout of the interface
- **JavaScript** – Game logic and functionality
- **LocalStorage** – Saving scores and comments in the browser

---

## Features

- Snake movement using keyboard arrow keys
- Random food generation
- Score tracking system
- Increasing game speed as score increases
- Obstacle mode for higher difficulty
- Pause and resume functionality
- Game Over screen with score saving
- Leaderboard using localStorage
- Forum section for comments and tips

## How to Run the Project

1. Download or clone the project folder.
2. Open the folder.
3. Open the file **index.html** in any web browser.

The game will start and you can begin playing.

---

## How to Play

- Use the **Arrow Keys** to control the snake.
- Eat the red food to increase your score.
- Avoid hitting:
  - Walls
  - Obstacles
  - The snake's own body
- When the game ends, enter your name to save your score in the leaderboard.
